Приветствую! Это клиент и сервер сделанный с помощью интерфейса WinSock2. С помощью сервера создаётся связь между двумя клиентами. В наших программах передаются текстовые сообщения. Из этого легко может быть реализована передача других данных, например положение курсора мыши. Тем самым может быть реализована игровая сессия для двоих в игру Shariki. 
Написанное выше планирутеся реализовать в ближайшее время. Удачи Вам и хорошего настроения.
